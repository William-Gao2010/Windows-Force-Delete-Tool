## Windows 文件夹强制删除助手 🛡️
一款由 Python 编写的轻量级工具，专门解决 Windows 下文件夹“权限不足”或“被程序占用”无法删除的问题。
## ✨ 功能特性 - **自动提权**：内置 UAC 请求，自动获取管理员权限。 

- **强制清理**：采用 \`shutil.rmtree\` 递归算法，强制移除只读文件和顽固目录。 
- **安全确认**：执行前需手动输入 \`y\` 确认，防止误操作。 
- **简单易用**：支持直接将文件夹拖入窗口识别路径。
## 🚀 快速开始 
1. 在 [Releases](https://github.com/William-Gao2010/Windows-Force-Delete-Tool/releases) 页面下载最新的 \`.exe\` 文件。 
2. 以管理员身份运行（程序会自动弹出请求）。
3. 拖入或输入要删除的文件夹路径，按回车并确认。 
⚠️ 警告 **本工具执行的是永久删除，文件不会进入回收站。请在确认前务必核对路径。** --- **作者**: [William-Gao2010](https://github.com/William-Gao2010)
